# AI-Powered Personal Fitness Companion

This capstone project focuses on the design and implementation of an AI-powered personal fitness companion. The application leverages advanced generative AI models to provide tailored workout routines, diet recommendations, and general wellness advice. The core functionality revolves around dynamic content generation based on user inputs and diverse prompt engineering strategies, enabling the system to adapt recommendations for various fitness goals such as fat loss, muscle gain, or maintenance.

The project emphasizes full-stack development without external database dependencies, running entirely on a local host environment. Students will integrate leading LLM APIs (Gemini or OpenAI) with a Python backend and develop a responsive frontend to create an interactive user experience. A key component involves designing and comparing multiple prompt engineering approaches to optimize the relevance and quality of AI-generated fitness plans.

The ultimate goal is to produce a functional, non-medical fitness assistant that demonstrates proficiency in AI integration, prompt engineering, and modern web application development. The project provides hands-on experience in building a practical AI application from scratch, addressing the need for accessible and personalized fitness guidance.
